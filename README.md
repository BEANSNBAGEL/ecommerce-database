# E-Commerce Database System

This repository contains the SQL implementation of an E-commerce database for the B103 Databases & Big Data module at Gisma University of Applied Sciences.

## Contents

- `sql/create_tables.sql` – SQL code to create all tables.
- `sql/insert_data.sql` – Sample data for testing.
- `sql/queries.sql` – CRUD and advanced queries (joins, aggregations).
- `diagrams/er_diagram.png` – ER diagram of the database.

## How to Run

1. Open a SQL environment (e.g., MySQL Workbench or dbfiddle.uk).
2. Run `create_tables.sql` to create the schema.
3. Run `insert_data.sql` to insert sample data.
4. Run queries from `queries.sql` to test functionality.
